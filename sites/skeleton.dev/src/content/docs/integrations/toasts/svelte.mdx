For most users we recommend the use of custom toast templates, which enable Tailwind styles.

```svelte
<!-- page.svelte -->
<script>
	import toast, { Toaster } from 'svelte-french-toast';
	import ToastError from './ToastError.svelte';

	function triggerError() {
		toast.custom(ToastError, { duration: 5000 });
	}
</script>

<!-- CustomToast.svelte -->
<!-- ToastError.svelte -->
<div class="card preset-filled-error-500 py-2 px-3 shadow-xl">
	<p class="text-xs">Uh oh, something went wrong.</p>
</div>

<Toaster position="bottom-right" />
```
