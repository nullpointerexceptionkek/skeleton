## Setup the Provider

Wrap the contents of your app with `ToastProvider` in your root layout. We'll interface with this using the [Svelte Context API](https://svelte.dev/docs/svelte/context).

```svelte title="+layout.svelte"
<script lang="ts">
	import { ToastProvider } from '@skeletonlabs/skeleton-svelte';
</script>

<ToastProvider>
	{@render children()}
</ToastProvider>
```

## Create a Toast

There are three available types of toasts: `info | error | success`, with `info` being the default.
